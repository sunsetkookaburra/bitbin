# Bit Bin

> Garbage In, Garbage Out!

Binary data encoding and decoding in a convenient TypeScript library.

## Available Data Types

`Uint8`, `Uint16`, `Uint32`, `BigUint64`, `Int8`, `Int16`, `Int32`, `BigInt64`,
`Bytes`, `Struct`, `Tuple`, `Vec`, `Utf8`
